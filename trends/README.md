# Trend Memory

This folder stores current trend signals by website/product area.

Use it when a project needs:
- current direction signals
- "what is trending now" context
- choices influenced by what platforms and markets are prioritizing
- a distinction between evergreen best practices and current movement

Core references:
- `trends/TRENDS-BY-AREA-2026.md`

Important:
- trend docs are time-sensitive
- they should be checked and refreshed periodically
- they are useful context, not a substitute for strong product fundamentals
