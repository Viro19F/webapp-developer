# Analytics Memory

This folder stores reusable measurement logic for websites and web apps.

Use it when a project needs:
- analytics setup
- event planning
- conversion measurement
- performance monitoring
- search visibility monitoring
- privacy-aware analytics choices

Core references:
- `analytics/MEASUREMENT-SPEC.md`
- `docs/ANALYTICS-MEASUREMENT-RESEARCH.md`

Why this exists:
- a website that cannot be measured cannot be improved
- future builds should inherit a stronger measurement model automatically
