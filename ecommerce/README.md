# Ecommerce Memory

This folder stores reusable conversion logic for ecommerce and retail websites.

Use it when a project needs:
- product pages
- category/search/filter logic
- cart and checkout decisions
- trust and conversion optimization
- first-time-buyer guidance
- specialty retail merchandising

Core references:
- `ecommerce/CONVERSION-SPEC.md`
- `docs/ECOMMERCE-CONVERSION-RESEARCH.md`

Why this exists:
- ecommerce sites win or lose in the product page, search, cart, and checkout details
- future store builds should inherit proven conversion patterns instead of starting from visual taste alone
