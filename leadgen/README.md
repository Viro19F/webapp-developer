# Lead Generation Memory

This folder stores reusable conversion logic for service-business websites and landing pages.

Use it when a project needs:
- more calls, bookings, or quote requests
- landing pages
- service business websites
- mobile-first lead capture
- form optimization
- consultative sales or audit-led proposals

Core references:
- `leadgen/CONVERSION-SPEC.md`
- `docs/LEAD-GENERATION-RESEARCH.md`

Why this exists:
- service sites convert differently than ecommerce stores
- future lead-gen builds should inherit better CTA, trust, form, and sales logic automatically
